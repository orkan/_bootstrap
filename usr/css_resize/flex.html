  • flex: grow shrink base
  • flex: 1 == flex: 1 1 0 (can grow & shrink from 0px up)
  • flex: 2 == flex: 2 1 0 (can grow & shrink from 0px up, but will grow faster than other flex:1 items)
  • flex: auto == flex: 1 1 auto (can grow & shrink from min-content up)
  • flex-grow: proportionaly add available free space
  • flex-shrink: proportionaly remove missing space
  • flex-base: initial size of an item
  • flex-base: [0|5px|20%|100%] == min size from which grow or shring gets calculated
  • flex-base: auto == min-content
